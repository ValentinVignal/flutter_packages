# markdown_lite

A simple package that parses markdown text to an AST.

## Supported Markdown Elements

- [ ] Headings (H1-H6)
- [ ] Paragraphs
- [ ] Bold, italic and strike-through text
- [ ] Lists (ordered and unordered)
- [ ] Code blocks
- [ ] Links
- [ ] Inline code
- [ ] Blockquotes
- [ ] Checkboxes
